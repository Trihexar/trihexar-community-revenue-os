# Code of Conduct

Trihexar Community is an open-source project for builders, operators, contributors, and learners.

## Expected Behavior

- Be respectful and direct.
- Discuss ideas with technical clarity.
- Give useful feedback.
- Keep communication professional.
- Credit contributors fairly.
- Protect the project from spam, scams, and misleading claims.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Scam promotion or misleading financial claims.
- Spam, phishing, or wallet-draining content.
- Doxxing or exposing private information.
- Disruptive behavior that blocks project progress.

## Enforcement

Maintainers may remove content, close issues, reject pull requests, or block contributors who violate these rules.
