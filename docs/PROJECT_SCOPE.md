# Project Scope

## What This Project Is

This is a static open-source starter website for Trihexar Community.

It includes:

- Public landing page
- Logo asset integration
- Modern responsive UI
- Vanilla JavaScript interactions
- README and contribution documents
- Basic brand and scope documentation

## What This Project Is Not

This project does not include:

- Smart contracts
- Wallet connection
- Token issuance
- Payment processing
- User database
- Backend API
- Investment product
- Financial advice

## Recommended Next Technical Steps

1. Publish the repository on GitHub.
2. Enable GitHub Pages.
3. Add issue templates.
4. Add a `/docs` navigation page.
5. Add a pricing or services page.
6. Add a contributor onboarding page.
7. Add analytics after privacy review.
8. Add a backend only when there is a clear operational need.
