# Trihexar Community Brand Guide

## Brand Positioning

Trihexar Community is a Web3 Community Revenue OS. It should feel technical, disciplined, modern, and builder-focused.

## Core Message

Build a community that can:

- Attract builders.
- Organize contributors.
- Document operations.
- Create revenue systems.
- Prepare for Web3 governance without speculative claims.

## Visual Direction

Current visual identity:

- Deep navy background
- Gold/copper accent
- Glass UI surfaces
- Sharp typography
- Dashboard-like components

## Tone

Use practical, operational language.

Preferred words:

- Build
- Ship
- Revenue operations
- Community infrastructure
- Governance
- Contributors
- Open source
- Documentation

Avoid words that create legal or trust issues:

- Guaranteed profit
- Risk-free income
- Token pump
- Investment return
- Financial promise
